O que é ?

O MySQL é um sistema de gestão de base de dados (SGBD), que utiliza a linguagem SQL (Linguagem de Consulta Estruturada, do inglês Structured Query Language) como interface. É atualmente um dos bancos de dados mais populares , com mais de 10 milhões de instalações pelo mundo.

É usado por muitas empresas de grande nome, entre elas:

Criado na Suécia por dois suecos David Axmark e Allan Larsson e um finlandês Michael "Monty" Widenius, que têm trabalhado juntos desde a década de 1980.

Atualmente o seu desenvolvimento e manutenção emprega aproximadamente 400 profissionais no mundo inteiro, e mais de mil contribuem testando o software, integrando-o em outros produtos e escrevendo documentos técnicos.

Em 2008, a MySQL AB empresa que iniciou o desenvolvimento do MySQL, foi adquirida pela Sun Microsystems, por US$ 1 bilião, um preço nunca antes atingido na área do software de licenças livres.

Em 2009, a Oracle compraria a Sun Microsystems e todos os seus produtos, incluindo o MySQL.

O sucesso do MySQL deve-se em grande medida à fácil integração com o PHP permitindo criar práticamente todo o tipo de aplicações para a internet. Além das empresas acima indicadas, o Facebook, Twitter e Wikipedia são exemplos de utilização do MySQL em sites de enorme transação de dados.

  • É um sistema que se utiliza na web;
  • É um sistema que corre num servidor;
  • Funciona bem tanto em pequenas como em grandes aplicações;
  • É muito rápido, fiável e fácil de utilizar;
  • Utiliza o SQL padrão;
  • É gratuito para download e utilização;
  • É desenvolvido, distribuído e suportado pela Oracle Corporation;

As bases de dados são úteis porque permitem armazenar a informação de uma forma lógica e organizada.

O MySQL baseia-se no modelo de bases de dados relacionais, isto é, os dados são armazenados em tabelas.

Uma tabela é uma coleção de dados relacionados que ficam estruturados em linhas e colunas.

Uma empresa pode ter que guardar informação dos seus funcionários, clientes, encomendas, etc. No sistema relacional cada uma dessas entidades transforma-se numa tabela e cada uma das tabelas vai se relacionar com outras para que toda a informação necessária possa ser consultada em qualquer instante.
